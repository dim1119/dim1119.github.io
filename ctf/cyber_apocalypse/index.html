Serial Logs⌗

We get a .sal file, so we must use logic analyzer 2.0

We notice 2 channels, so it must be a 2 channel protocol. When searching for raspberry pi debugging interface we find this site https://gr.mouser.com/new/ftdi-chip/ftdi-raspberry-pi-interface-modules/. It says that the pi offers a UART output.

We can use the autobaud function to get the baud rate.

The baud rate is 117925.

Lets use python to parse the data:

Secure⌗

The scenario mentions captured traffic from an sd card while in transit, probably used this one sd card module.

The SD card modules uses SPI.

  • Channel 0 is Master In Slave Out.
  • Channel 1 is Master Out Slave In.
  • Channel 2 is chip select.
  • Channel 3 is clock.

The flag is in MISO.

Off the grid⌗

We discover 6 channels.

OLED SH1306 uses SPI/I2C, in this case it is SPI.

  • D1 is clock
  • D2 is Chip Select
  • D0 is MOSI
  • D3 is MISO

Export the data and save the “MISO” column signal using awk and run:

from matplotlib import pyplot as plt
import bitmap
import re
from PIL import Image, ImageDraw
import numpy as np


file = open("miso.txt", "r")
pattern = "0xB[0-7]"

matrx = np.zeros(shape=(128, 64))

dc = 0
counter = -1
page = 0
page_counter = 0
sequence = []
data = []
for line in file:
    line = line.rstrip("\n")
    sequence.append(line)
    if dc == 0:
        if re.match(pattern, line):
            sequence = [line]
        elif re.match(pattern, sequence[0]) is not None and line == "0x02":
            sequence.append(line)
        elif re.match(pattern, sequence[0]) is not None and sequence[1] == "0x02" and line == "0x10":
            sequence.append(line)
            dc = 1
            counter = 0
    else:
        # reads data
        if counter == 128:
            counter = 0
            dc = 0
        else:
            data.append(line)
            counter += 1
file.close()

x_pos = 0
y_pos = 0
count = 0
page = 0
screen = 0
for num in data:
    bits = str(bin(int(num, 16))[2:].zfill(8))[::-1]
    for bit in bits:

        try:
            matrx[x_pos][y_pos] = bit
        except:
            exit("Exited! {} and {} with bit {}".format(x_pos,y_pos,bit))
            
        if y_pos%8 ==  7 :
            if x_pos == 127:
                print("End of Page {}:{}".format(x_pos,y_pos))
                if page == 7:
                    print("End of Screen")
                    plt.imshow(matrx[::-1], interpolation='nearest')
                    plt.gray()
                    plt.savefig(str(screen)+".png")
                    matrx.fill(0)
                    x_pos=0
                    y_pos=0
                    screen+=1
                    page=0
                else:
                    page += 1
                    x_pos = 0
                    y_pos = 0+page*8
            else:
                x_pos += 1
                y_pos = 0+page*8
                #print("End of column")
        else:
            y_pos += 1


    count += 1
print("{} of {} bytes\n{} screens".format(count, len(data),screen+1))
